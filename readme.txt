== Description ==

Accept Visa/Mastercard payments in your WooCommerce shop using TBC gateway.

= Free =
* SMS transaction type allows you to capture funds instantly
* Accepts GEL
* Close business day
* Autogenerated OK/FAIL urls
* Detailed logs
* 10 transactions per month

= Premium =
* All features from the Free version
* DMS transaction type allows you to block required amount first and capture funds later (within 30 days)
* Refunds
* Multiple TBC merchant accounts
* Accepts GEL, USD, EUR
* Unlimited transactions
* Support (mail/phone)

Learn more about our [premium plugin](https://plugandpay.ge/product/woocommerce-tbc-credit-card-payment-gateway/?utm_source=readme&utm_medium=premiumLinkDescription&utm_campaign=tbcfree)
